# Spindexer Subsystem

## Dependencies

Requires a motor that can control its duty cycle and the ability to stop.

## Public APIs

### Commands

#### indexBallIntoLauncher

`indexBallIntoLauncher` will index fuel into the launcher so that the launcher can shoot the ball.

`indexBallIntoLauncher` doesn't require any parameters.

### Triggers

#### isMotorTurning

`isMotorTurning` will check if the motor is spinning/moving.
